Teacher Dashboard

Every record saved or submitted on this device is collected below. Link a Google Form / Apps Script Web App URL to have student responses POSTed to your server automatically. This URL is shared with the ShirtpapaSG Genie — set it once and both apps log to the same sheet, with a Game Type column showing which app each row came from.

📘 First time? Set up your free Google Sheet server — step by step
  1. Go to sheets.google.com and create a new blank spreadsheet (this is where responses will land).
  2. In the menu, click Extensions ▸ Apps Script. A code editor opens in a new tab.
  3. Delete any sample code in the editor, then paste the script below (use the Copy button).
  4. Click the Save icon 💾 and give the project any name.
  5. Click Deploy ▸ New deployment (top-right).
  6. Click the gear ⚙ beside “Select type” and choose Web app.
  7. Set Execute as: Me and Who has access: Anyone, then click Deploy.
  8. Click Authorize access → pick your Google account → Advanced → Go to project (unsafe) → Allow.
  9. Copy the Web app URL shown — it ends in /exec.
  10. Paste that URL into the Server endpoint box above, click Save URL, then press Test. A new row should appear in your Google Sheet. ✅
